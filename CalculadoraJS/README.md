####CalculadoraJS:
<br />

Projeto do curso online utilizando apenas JavaScript
<br />

Recursos interessantes da calculadora :
  * Áudio - clicar duas vezes AC para ativar; 
  * Eventos de teclado;
  * CE - limpa o último número digitado;
  * Data e Hora; <br />
 
Alguns métodos utilizados no projeto: 
  * split();
  * eval();
  * join();
  * toString();
  * forEach();
  * condicional switch;
